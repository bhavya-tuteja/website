Prerequisits for React?

  1. Basic knowledge of HTML, CSS and JavaScript.
  2. Basic understanding of ES6 features
    1. ES6 means- ECMAScript6(2015)
  3. Basic understanding of how to use npm( note-packing manager)
  4. VS Code( Visual Studio Code) to be downloaded.
    1. Delete Welcome from VS Code
    2. First Icon: Explorer - Open Editors, No folder opened, Outline, Timeline
    3. Second Icon: Search icon to serach some file or folder.
    4. Third Icon: Source Control- In order to use GIT
    5. Fourth Icon: Run and Debu-To customize Run And Debug, open a folder.
    6. Fifth Icon: Extensions - Use it for help and simplification of code.
