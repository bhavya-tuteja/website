SIMPLE SELECTORS

FIRST TYPE - It is first category. This categaor is furthur divided in five selectors:

  • ELEMENT SELECTOR
  • ID SELECTOR
  • CLASS SELECTOR
  • UNIVARSAL SELECTOR
  • GROUPING SELECTOR

ELEMENT SELECTOR In this type - If in the type we can we use HTML TAGS such as 'h1', 'h2' and 'p' tag.
We will handle 'h1', 'h2' and 'p' tags by giving the followin commands
'h1 { color: red}', 'h2{color:green}' and 'p{color:blue} in css file'

ID SELECTOR In this type we assign an 'id' within a tag
For example 'div id="box'. In css file we assign '#' sign as follows
'#box{ width: 10 rem; height: 10rem; background-color: red;' is the command in css file.

It is a test of Id selector. In this '# is assigned before the 'id'. The dimentions of div are width: 20 rem and height: 20rem

CLASS SELECTOR In this type we assign a class . It is done in say a div 'class=box-class.
In css file we assign class with a dot as follows
'div.box-class{ width: 20rem; height: 20rem; background-color: red;''

It is test of class-selector. In the css a dot is assigned before the class
It is as follows: { width: 20rem; height: 20rem; background-color: blue}

UNIVERSAL SELECTOR: In this type we assign '*' and { }. If we write any command. It is a Universal Command
It will be applied to all the tags. For Example: '*{ margin:0; padding:0;}
It is to be written in css file.
